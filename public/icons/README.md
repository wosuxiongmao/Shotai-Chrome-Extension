# Extension Icons

Place your extension icons here:

- `icon-16.png` - 16x16 (browser toolbar)
- `icon-48.png` - 48x48 (extension management page)
- `icon-128.png` - 128x128 (Chrome Web Store, installation)

## Design Guidelines

- Use the ShotAI brand colors and logo
- Icons should work on both light and dark backgrounds
- Use PNG format with transparency
- Follow Chrome's icon design guidelines

## Tools

- [Figma](https://figma.com) - Design icons
- [TinyPNG](https://tinypng.com) - Compress PNGs
- [Photopea](https://www.photopea.com) - Free online editor
